# MSPR Bloc 4
## Client service

Ce service a vocation de gérer les opérations client, authentification client, admnistration, etc.

### Lancement du projet

**Prérequis**

- Avoir docker ou podman sur sa machine

**Développement**  

Ouvrez la solution dans un IDE (Visual Studio, Rider) puis lancez le projet AppHost.  
Depuis VSCode, il vous faudra la cli dotnet ```dotnet run --project src/Kawa.ClientService.AppHost```
